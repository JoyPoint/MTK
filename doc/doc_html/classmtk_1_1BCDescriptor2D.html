Detailed Description

This class presents an interface for the user to specify boundary conditions on 2D mimetic operators and the grids they are acting on.

Def. Let be any scalar or vector field defined over a domain . We can specify any linear combination of and its derivatives to fulfill a condition, which we define as a boundary condition:

This class receives information about all possible coefficient functions, for any subset of the boundary (south, north, west and east), and each condition for any subset of the boundary, and takes care of assigning them to both, the differentiation matrices and the grids. The highest order of differentiation is computed based on how many coefficients have been given at run-time.
