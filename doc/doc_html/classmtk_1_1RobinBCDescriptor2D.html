Detailed Description

This class presents an interface for the user to specify Robin boundary conditions on 2D mimetic operators and the grids they are acting on.

Def. Let be the solution to an ordinary or partial differential equation of interest. We say that satisfies a Robin boundary condition on if and only if there exists so that:

Intuitively, a Robin boundary condition is a constraint that must be satisfied by any linear combination of any scalar field and its first normal derivative, in order for to represent a unique solution to a given ordinary or partial differential equation of interest.

Instances of this class receive information about the coefficient functions and each condition for any subset of the boundary (west, east, south and north in 2D). These instances then handle the complexity of placing the coefficients in the differentiation matrices and the conditions in the grids.
